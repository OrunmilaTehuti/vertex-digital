# Vertex Digital Partners 🚀

Welcome to the official repository for **Vertex Digital Partners**—a premium digital solutions agency engineered to help modern businesses build, optimize, and scale their online presence. 

This repository contains the complete, lightweight source code for our responsive corporate website, built using modern web standards and hosted entirely on GitHub Pages.

---

## 🛠️ Technology Stack
To ensure maximum performance, clean code architecture, and fast load times, this website is built with:
* **HTML5:** Semantic structural layout.
* **Tailwind CSS:** Modern utility-first styling with custom glassmorphism components.
* **JavaScript (Vanilla):** Light, efficient interactions (mobile nav toggle, dynamic scroll animations).
* **FontAwesome:** Scalable vector icon integration.
* **Google Fonts:** Premium typography pairing (*Space Grotesk* for headings, *Inter* for body text).
